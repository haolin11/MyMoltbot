# 🎨 方案书Canvas画布升级

**更新时间**: 2026-01-29  
**版本**: Canvas Edition v1.0  
**设计灵感**: ChatGPT Canvas

---

## 🎯 设计目标

将方案书页面从传统的卡片布局改造为**专业文档画布**视图，提供类似ChatGPT Canvas的文档编辑体验。

---

## ✨ 核心改进

### 1. **Canvas画布布局** 📄

**改进前**：
- ❌ 传统网页卡片布局
- ❌ 左右分栏，视觉分散
- ❌ 缺乏文档感
- ❌ 不适合长文档阅读

**改进后**：
- ✅ 白色A4文档画布居中显示
- ✅ 灰色背景衬托文档区域
- ✅ 阴影效果模拟纸张悬浮
- ✅ 专业的文档排版

```css
.canvas-document {
  background: white;
  box-shadow: 0 0 0 1px rgba(0,0,0,0.05), 
              0 2px 8px rgba(0,0,0,0.08), 
              0 8px 32px rgba(0,0,0,0.06);
  border-radius: 8px;
  max-width: 900px;
  padding: 3rem 4rem;
}
```

---

### 2. **右侧工具栏** 🛠️

**功能清单**：
- ✅ **导出为PDF** - 直接打印/保存
- ✅ **导出为Word** - 开发中
- ✅ **复制全文** - 一键复制到剪贴板
- ✅ **返回编辑** - 跳转到生成页面
- ✅ **分享方案** - 开发中
- ✅ **文档大纲** - 快速导航

**设计特点**：
- 固定在右侧，不随页面滚动
- 简洁的图标+文字按钮
- Hover悬停效果
- 320px宽度，不遮挡文档

---

### 3. **专业文档排版** 📝

#### 标题层级体系

| 级别 | 字号 | 字重 | 用途 |
|------|------|------|------|
| **文档标题** | 2.5rem (40px) | 700 | 方案名称 |
| **一级标题** | 1.75rem (28px) | 700 | 章节标题 |
| **二级标题** | 1.25rem (20px) | 600 | 小节标题 |
| **正文** | 1rem (16px) | 400 | 段落内容 |
| **辅助文字** | 0.875rem (14px) | 400 | 说明文字 |

#### 间距系统
- **章节间距**: 3rem (48px)
- **段落间距**: 1rem (16px)
- **行高**: 1.8（正文）
- **文档边距**: 3rem 4rem（上下 左右）

#### 颜色系统
- **主标题**: #111827（深灰黑）
- **正文**: #374151（中灰）
- **辅助**: #6B7280（浅灰）
- **卡片背景**: #F9FAFB（极浅灰）
- **边框**: #E5E7EB（浅灰边框）

## 🚀 技术实现

### HTML结构
```html
<div class="flex">
  <!-- 主文档区 -->
  <main class="canvas-container">
    <article class="canvas-document">
      <!-- 文档内容 -->
    </article>
  </main>
  
  <!-- 右侧工具栏 -->
  <aside class="canvas-sidebar">
    <!-- 工具按钮 -->
  </aside>
</div>
```

### 核心CSS
```css
/* 文档画布 */
.canvas-document {
  background: white;
  box-shadow: 0 0 0 1px rgba(0,0,0,0.05), 
              0 2px 8px rgba(0,0,0,0.08), 
              0 8px 32px rgba(0,0,0,0.06);
  max-width: 900px;
  padding: 3rem 4rem;
}

/* 工具栏 */
.canvas-sidebar {
  position: sticky;
  top: 64px;
  height: calc(100vh - 64px);
  width: 320px;
}
```

### JavaScript功能
```javascript
// 复制全文
function copyToClipboard() {
  const doc = document.querySelector('.canvas-document');
  navigator.clipboard.writeText(doc.innerText);
}

// 导出PDF
window.print(); // 直接调用浏览器打印功能
```

## 📝 使用说明

### 查看方案书
1. 在协作生成页面完成方案
2. 点击"预览"按钮
3. 进入Canvas文档视图

### 导出PDF
1. 点击右侧工具栏"导出PDF"
2. 或点击顶栏"导出PDF"按钮
3. 浏览器打印对话框打开
4. 选择"保存为PDF"
5. 保存到本地

### 复制全文
1. 点击右侧工具栏"复制全文"
2. 文档内容已复制到剪贴板
3. 可粘贴到任何地方

### 返回编辑
1. 点击"返回编辑"按钮
2. 跳转到协作生成页面
3. 继续修改方案内容
