## 🎨 新配色方案

### 核心色彩

| 颜色类型 | 旧版本 | 新版本 | 说明 |
|----------|--------|--------|------|
| **主色** | #2563EB 蓝色 | #1E293B 深石板灰 | 专业稳重 |
| **辅助色** | #7C3AED 紫色 | #10B981 翡翠绿 | 信任可靠 |
| **强调色** | - | #F59E0B 琥珀橙 | 活力引导 |
| **风格** | 蓝紫渐变 | 深色纯色/微妙渐变 | 克制高级 |

### 配色原则
1. **60-30-10法则**
   - 60% 深石板灰（主色）
   - 30% 白色和浅灰（背景）
   - 10% 翡翠绿+琥珀橙（强调）

2. **避免渐变**
   - 减少渐变色使用
   - 使用纯色或微妙色调变化
   - 更加克制和专业

3. **功能色优先**
   - 翡翠绿 → 成功/在线/积极
   - 琥珀橙 → 重要/警告/引导
   - 红色 → 错误/危险
   - 蓝色 → 信息提示

## 👨‍💼 专家协作优化

### 改进前的问题
- ❌ "专家协作"功能位置不明确
- ❌ 与"引用/证据"逻辑分离
- ❌ 缺少清晰的申请入口

### 改进后的方案

#### 1. **整合位置**
```
引用/证据标签页
├── 专家协作卡片（顶部）
│   ├── 专家头像 + 在线状态
│   ├── 功能说明
│   └── "申请专家协作"按钮（琥珀橙）
├── 引用/证据列表
│   ├── 专家提供的引用（琥珀橙标识）
│   └── 其他引用
└── 快速添加工具栏
    └── 包含"请教专家"按钮
```

#### 2. **专家卡片特点**
- ✅ 渐变背景（翡翠绿 → 琥珀橙）
- ✅ 专家头像 + 在线/离线状态点
- ✅ 清晰的功能说明
- ✅ 醒目的申请按钮（琥珀橙）

#### 3. **引用标识**
- 专家提供的引用：琥珀橙边框
- "专家提供"徽章：琥珀橙背景
- 渐变背景突出显示

## 🎨 配色效果对比

### 主要按钮

**改进前**：
```css
background: linear-gradient(135deg, #2563EB, #7C3AED);
color: white;
```
- 蓝紫渐变
- 过于常见
- 缺乏辨识度

**改进后**：
```css
/* 主要按钮 */
.cp-btn-primary {
  background: #1E293B;  /* 深石板灰 */
  color: white;
}

/* 成功按钮 */
.cp-btn-success {
  background: #10B981;  /* 翡翠绿 */
  color: white;
}

/* 强调按钮 */
.cp-btn-accent {
  background: #F59E0B;  /* 琥珀橙 */
  color: white;
}
```
- 深色调专业
- 功能性明确
- 高级克制

## 👨‍💼 专家协作交互流程

```
1. 用户点击"引用"标签页
   ↓
2. 看到"专家协作"卡片
   - 专家头像（带在线状态点）
   - 功能说明文字
   - "申请专家协作"按钮（琥珀橙）
   ↓
3. 点击"申请专家协作"
   ↓
4. 按钮显示loading
   "正在连接专家..."
   ↓
5. 连接成功
   - 按钮变为"专家已加入"（翡翠绿）
   - 引用列表新增专家欢迎消息
   - 带有"专家提供"徽章（琥珀橙）
   ↓
6. 用户可继续提问
   - 专家回复自动添加到引用列表
   - 所有专家提供的引用都有琥珀橙标识
```
